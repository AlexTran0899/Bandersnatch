# Bandersnatch: Data Science Monster App

### Components
- Logic: Python3
- API Framework: Flask
- Templates: Jinja2
- Structure: HTML5
- Styling: CSS3
- Database: MongoDB
- Graphs: Altair
- Machine Learning: Scikit
- Hosting: Heroku

### Features
- Data Creation
    - Random
    - Custom
- Data Viewing
    - Raw Data
    - Interactive Graph
- Model Training
- Rank Prediction
    - Input Features => Predicted Target

### Monster Fields
- Name
- Type
- Level
- Rank
- Damage
- Resources
    - Health
    - Energy
    - Sanity
